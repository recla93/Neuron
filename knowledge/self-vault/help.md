# help — indice rapido dei comandi Neuron

Mostra tutti i comandi Neuron in formato compatto (una riga ciascuno) più una breve guida su come usare Neuron correttamente.

## Quando usarlo

- Primo comando se non sai cosa fare o quali tool esistono.
- Per avere l'elenco completo prima di chiamare [[skill]] per il dettaglio.
- Punto di partenza per qualunque sessione con Neuron.

## Come si usa

```
help()
```

Nessun parametro. Ritorna l'indice dei comandi e il flusso consigliato.

## Vantaggio

Risposta immediata, zero side effect. È l'entry point informativo.
